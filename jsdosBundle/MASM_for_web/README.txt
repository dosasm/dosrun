This bundle is a package of MASM Tools
Microsoft Macro Assembler(MASM)

- MASM 5.00
- link 3.60

# script for Run Code

```Run
masm test.asm;
link test.obj;
test.exe
```

# script for Debug Code

```Debug
masm test.asm;
link test.obj;
debug test.exe
```

# script for show list

```list
masm /la test.asm;
```
